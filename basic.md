## 2. 기호 *
#### * : 글씨 기울기,굵기 조정
 1. 기울기는 * 1개
It is *italic*  
 2. 굵기는 ** 2개 
It is **bold**  
 3. 기울기+굵기는 ***3개
It is ***italic and bold***  
 4. 취소선은 ~~
It is ~~cancelled~~
